项目自评表

为了方便测试，编写了一个简单的makefile

```makefile
FILENAME=ex_prime3.c
arg=8
run: clean
	dotnet run --project interpc.fsproj example/$(FILENAME) $(arg)

debug: clean
	dotnet run --project interpc.fsproj -g example/$(FILENAME) $(arg)

clean:
	dotnet clean interpc.fsproj
```

运行解释器

```bash
# 直接运行
make
# 以调试模式运行
make debug
# 添加参数
make debug arg=8
```
